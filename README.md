### What I'm building

I design and build the **Workiva AI ecosystem** — a suite of tools that connects LLMs to enterprise document APIs:

| Project | What it does |
|---------|-------------|
| **workiva-sdk** | Python SDK for Workiva APIs — Platform, Chains & Wdata |
| **workiva-mcp** | MCP server that exposes Workiva APIs as tools for LLM agents |
| **workiva-ai** | Conversational AI agent for document management (multi-provider LLM, SSO/OIDC) |
| **workiva-scripts** | Automation scripts for Workiva clients |
